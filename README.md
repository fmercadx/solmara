# Solmara: Private Wellness Atelier (Concept)

A portfolio design study by **Francis Mercado**: a premium wellness / spa-luxury brand concept with an editorial marketing site and a working client-onboarding flow.

**Solmara is a fictional business.** All names, people, testimonials, and details are invented. No orders are taken and no data is collected; the onboarding flow is a front-end demonstration only (nothing entered leaves the browser; payment is simulated, and in a production build would run on Stripe).

## Pages

- `index.html`: Home
- `how-it-works.html`: the four-movement method
- `about.html`: atelier philosophy + founder
- `faq.html`: accordion FAQ
- `begin.html`: onboarding flow (intake questionnaire, photo upload with drag & drop previews, simulated checkout, confirmation)

## Stack

Hand-built static HTML/CSS/JS with no frameworks and no build step. Bodoni Moda + Hanken Grotesk via Google Fonts. Deployable to any static host (GitHub Pages).

## Design notes

Quiet-luxury editorial direction: high-contrast didone display type, warm limestone palette with terracotta and eucalyptus accents, arch motifs, film-grain texture, staggered load-in and scroll reveals.
